Fitting a power law to RBN data¶

This file processes the MergedList.csv file that is produced by the output_summary_combine.m script and contains upper and lower bounds for the number of attractors for the RBN networks in the ensemble. It also processes the MergedListTR_full.csv file that, in addition to the MergedList.csv data, contains lower bounds on the attractor counts calculated by finding minimal trap spaces using our iterative reduction methods.

We begin by importing various Python libraries we will need and setting plotting defaults. This notebook was tested by the authors using Python version 3.8.3 in Jupyter notebook version 6.0.3.

We generate the functions that will be used for the fitting. We are fitting the function $\langle A\rangle = a + bN^c $, and need to calculate its Jacobian with respect to the parameters $x=(a,b,c)$.

def scaling_fit(x, u):
    ## <A> = a + bN^c , x=(a,b,c);<A>=y;N=u
    return x[0] + x[1] * (u ** x[2]) 

def fun(x, u, y):
    return scaling_fit(x, u) - y

def jac(x, u, y):
    ##  <A>_a = 1
    ##  <A>_b = N^c
    ##  <A>_c = b*ln(N)*N^c
    J = np.empty((u.size, x.size))
    powerlaw = u ** x[2]
    J[:, 0] = 1 
    J[:, 1] = powerlaw
    J[:, 2] = x[1] * powerlaw * np.log(u)
    return J

We will fit the RBN data using nonlinear least-square fitting.

For each $N$, the attractor number $A$ is an independent random variable distributed according to $P(A;N)$. We generate an ensemble $\Omega(N)$ for each $N$, and estimate the average of $A$ for this ensemble: $\langle A\rangle_{\Omega(N)}$. The difference between $\langle A\rangle_{\Omega(N)}$ and the true $\langle A\rangle$ per $N$ should be an independent and normally distributed random variable as $\Omega(N) \rightarrow \infty$ (i.e. central limit theorem should hold for the estimate of $\langle A\rangle$ for each $N$).

Loading the RBN data¶

We apply the following strategy for the networks with very high upper bounds and the ones for which we could not find bounds on the number of attractors:

  • In some cases, the upper bound, $U$, is very large -- these are theoretical bounds obtained by considering an STG made of only 2-cycles and are extremely generous upper bounds. We consider $U$ to be unknown if it is greater than the lower bound, $L$, by more than 100.
  • If $L$ is missing, set it to 1. This is a universally valid lower bound, as ever network must have at least one attractor.
  • If $U$ is missing and $L$ is known, set $U$ to $1.25\times L$. This is equivalent to the assumption that, on average, each minimal trap space contains 1.25 attractors. Based on the experience of the authors, this is much higher than the likely (unknown) value.
  • If $U$ is missing and $L$ is unknown, we first set $L=1$. To set $U$ for the $m_N$ intractible networks for each $N$, we select the $m_N$ largest known values of $U$ for networks of size $N$ (for which $U-L\leq 100$). We increase these values by $10\%$ and assign them to the intractible networks. This ensures that these networks have relative high upper bounds that are not overly sensitive to potential high-$U$ outliers.

For data in MergedListTR_full, we only use the lower bounds, as these networks were not analyzed with the goal of obtaining upper bounds.

We next attempt to fit a power law to each of these data sets. As our initial guess we use $\langle A\rangle=a + bN^c$ with c=0.5 (the original powerlaw from Kauffman) and $a$ and $b$ such that the values at the extremes ($\langle A\rangle_{N=2}$ and $\langle A\rangle_{N=4096}$) are exactly fit. If the parameters for the resulting initial fit fall outside reasonable bounds, we fall back to $\langle A\rangle=N^{1/2}$. We then use SciPy to perform a local least-squares optimization with the chosen values of $a$, $b$, and $c$ as the initial point.

def fit_to_data(u_vals,y_vals,verbose=1,bounds = ([-10,0,0],[10,10,2])):
    u = np.array(u_vals)
    y = np.array(y_vals)

    a_num=np.sqrt(u[0])*y[-1]-np.sqrt(u[-1])*y[0]
    b_num=y[0]-y[-1]
    ab_dem=np.sqrt(u[0])-np.sqrt(u[-1])
    a=a_num/ab_dem
    b=b_num/ab_dem

    x0 = np.array([a,b,0.5])
    
    try:
        res = least_squares(fun, x0, jac=jac, bounds=bounds, args=(u, y), verbose=verbose)
    except ValueError:
        x0 = np.array([0,1,0.5])
        res = least_squares(fun, x0, jac=jac, bounds=bounds, args=(u, y), verbose=verbose)
        
    exponent="{0:.3g}".format(res.x[2])
    return(res,exponent)

Confidence interval of exponents through boostrapping¶

To find the confidence intervals of the fits, we will use a boostrap approach.

  • For each $N$, a boostrap sample is a sample with replacement from $\Omega(N)$ of size $\left|\Omega(N)\right|$.
  • For each boostrap sample, we use nonlinear least squares to fit the curve and get the best-fit exponent.
  • We generate $n_{boostrap}=10000$ bootstrap samples and compute the best-fit exponent for each. From these fits, we obtain the 95% confidence interval of the exponent from the best-fit exponent distribution ($2.5\%$ and $97.5\%$ quantiles).
  • We will see that the bootstrap distributions are approximately normal, and so we will also compute their standard deviations.

We begin by defining the relevant bootstrap functions.

def bootstrap_generate(df_data,n_bootstrap,seed):
    rd.seed(seed)
    N_vals=list(df_data["N"].unique())
    A_means={};UA_means={};LA_means={}
    
    for n in df_data["N"].unique():
        df_n=df_data[df_data["N"]==n]
        UA_mean=[];LA_mean=[]  
        for n_rep in range(n_bootstrap):    
            resample=rd.choices(df_n["U"].tolist(),k=df_n["U"].count())
            UA_mean.append(np.mean(resample))
            
            resample=rd.choices(df_n["L"].tolist(),k=df_n["L"].count())
            LA_mean.append(np.mean(resample))
        
        df_n=df_n.loc[df_n["U-L"]==0]
        A_mean=[];
        for n_rep in range(n_bootstrap):
            resample=rd.choices(df_n["U"].tolist(),k=df_n["U"].count())
            A_mean.append(np.mean(resample))
            
        A_means[n]=A_mean
        UA_means[n]=UA_mean
        LA_means[n]=LA_mean
    return(A_means,UA_means,LA_means)

def bootstrap_generate_tr(df_data,n_bootstrap,seed):
    rd.seed(seed)
    N_vals=list(df_data["N"].unique())
    LA_means={}
    
    for n in df_data["N"].unique():
        df_n=df_data[df_data["N"]==n]
        LA_mean=[]  
        for n_rep in range(n_bootstrap):
            resample=rd.choices(df_n["L"].tolist(),k=df_n["L"].count())
            LA_mean.append(np.mean(resample))
        
        LA_means[n]=LA_mean
    return LA_means

def bootstrap_fit(bootstrap_data,n_bootstrap):
    u = np.array(list(bootstrap_data.keys()))
    fits=[]
    exponents=[]
    sk = 0
    for n_rep in range(n_bootstrap):
        y = np.array([bootstrap_data[n][n_rep] for n in u])
        res,exponent=fit_to_data(u,y,verbose=0)
        fits.append(res)
        exponents.append(res.x[2])
    
    return(fits,exponents)
